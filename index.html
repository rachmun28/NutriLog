Google Sheets setup

One Apps Script handles the food log and custom foods database. It supports two-way sync — pulls your history on load.

  1. Open Google Sheets → create a new spreadsheet.
  2. Click Extensions → Apps Script. Delete existing code, paste the script below.
  3. Deploy → New deployment → Web app.
    Execute as: Me. Access: Anyone.
  4. Copy the Web App URL → paste in NutriLog Data screen → Save & connect.
