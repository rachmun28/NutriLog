Google Sheets setup

This single Apps Script handles both the Food Log and Custom Foods sheets automatically.

  1. Open Google Sheets → create a new spreadsheet.
  2. Click Extensions → Apps Script. Delete existing code, paste the script below.
  3. Deploy → New deployment → Web app. Execute as: Me. Access: Anyone. Copy the URL.
  4. Paste the URL in NutriLog → Settings → Apps Script URL.
